## Launch:

> - `cd backend/`
> - `npm install && npm run build && npm start`

> - `cd frontend/`
> - `npm install && npm start`
> - `npm run desktop`
> - `npx cap copy`
> - `npx cap open android`
> - `npx cap open ios`
