---

## Launch:

> - `cd backend/`
> - `npm install && npm run build`
> - `npm start`

> - `cd frontend/`
> - `npm install && cordova platform add android && cordova platform add ios`
> - `npm run web`
> - `npm run desktop`
> - `npm run android`
> - `npm run ios`
